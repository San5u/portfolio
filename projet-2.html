Portfolio Sophie Bluel (Javascript)

Description: Développement d’un portfolio dynamique en JavaScript permettant une interaction fluide et une présentation des projets sous forme de galerie.

La principale problématique de ce projet était de transformer un site statique en un site dynamique, capable de gérer des opérations telles que l’ajout et la suppression de photos des projets de l’architecte d’intérieur. Le défi résidait dans l’intégration d’une modale intuitive pour uploader des médias, la création d’une page de connexion sécurisée pour l’administrateur, et l’utilisation des données provenant d’une base pour afficher les informations de manière dynamique.

Ce projet a permis de renforcer les compétences en JavaScript, notamment pour manipuler le DOM, gérer les événements, et mettre en place des fonctionnalités dynamiques comme la suppression et l’ajout de photos. La gestion des requêtes API a été essentielle pour interagir avec une base de données. L’utilisation des modales a également permis d’approfondir les notions de développement orienté expérience utilisateur.

Les technologies utilisées incluent principalement JavaScript pour rendre le site interactif, ainsi que les API pour communiquer avec le backend et gérer les données des projets. Le projet s’est appuyé sur HTML et CSS pour la structure et le style, mais le cœur du travail reposait sur la logique JavaScript et l’intégration des données dynamiques.
