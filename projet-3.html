Kasa, application web de location immobilière (React)

Description : Création d’une application web de location immobilière avec React, offrant une expérience utilisateur intuitive et des fonctionnalités avancées de gestion des biens.

Kasa, une entreprise spécialisée dans la location d’appartements entre particuliers, souhaite moderniser son site web développé il y a plus de 10 ans en ASP.NET. Cette refonte vise à adopter une nouvelle stack technologique axée sur React pour le front-end et Node.js pour le back-end. Le défi principal est de construire une interface dynamique, fonctionnelle et responsive à partir de maquettes Figma, tout en respectant les contraintes techniques telles que l'usage de JSON pour les données en l'absence d'un back-end opérationnel.

Développement d'applications React avec des composants modulaires. Implémentation des routes dynamiques avec React Router. Création d'une galerie d’images interactive avec des fonctionnalités avancées comme le défilement circulaire et l’adaptation à des listes d’images de tailles variables. Utilisation de Sass pour générer le style CSS et mise en place d’animations fluides, notamment pour les menus déroulants (Collapse). Lecture et utilisation de données statiques JSON pour simuler une API, en attendant l’intégration d’un back-end.

React : pour construire l’application front-end en suivant une architecture orientée composants. React Router : pour gérer les routes et assurer une navigation fluide entre les pages. Sass : pour créer un style modulaire et améliorer la gestion des animations. JSON : pour simuler une base de données et permettre l'affichage dynamique des annonces.
